# Roadmap

## Phase 0: Foundation

- Project structure
- Documentation
- Simulation types
- Static dashboard UI
- Figma concept

## Phase 1: Playable Tick

- Advance city time
- Needs decay
- Work, rest, and buy-food actions
- Economy event log
- Citizen personality controls

## Phase 2: Local AI Layer

- Ollama provider
- AI-generated city summary
- AI action proposals for selected citizen
- JSON validation and fallback behavior

## Phase 3: Persistence

- SQLite database
- Saved city snapshots
- Event history
- Citizen profile persistence

## Phase 4: Deeper Simulation

- Relationships
- Reputation effects
- Factions
- Crime/risk system
- Dynamic market prices
- Citizen memory

## Phase 5: Scenario Builder

- City presets
- Custom citizen creation
- Custom jobs and resources
- Import/export scenarios
