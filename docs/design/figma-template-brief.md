# Figma Template Brief

Figma file:

## Template Structure

### 01 Dashboard

Main simulation dashboard:

- app topbar
- large illustrated city map
- citizen inspector
- economy cards
- jobs list
- event log

### 02 Citizen Detail

Focused citizen profile:

- citizen portrait placeholder
- role, money, hunger, energy, reputation
- personality sliders
- trait chips
- behavior alignment preview
- recent actions

### 03 AI Proposal

Review state for local AI actions:

- AI proposed action
- affected citizen
- reason
- simulation validation result
- accept/reject controls

### 04 Assets

Reusable design pieces:

- map image
- citizen marker
- district label
- resource card
- event row
- slider row
- alignment chips
- color tokens

## Image Assets

- `public/city-map-modern.png`: current modern raster map artwork.

## Visual Quality Bar

The map should feel like a modern simulation product, not a placeholder. It
needs recognizable geography, depth, routes, neighborhoods, and live markers.
Use the modern raster city map as the base image, then layer only minimal UI
markers above it.
